\# 8-Puzzle Solver (BFS)



\## Опис проєкту



Ця програма реалізує алгоритм пошуку в ширину (Breadth-First Search, BFS) для розв’язання задачі гри в 8-пазл.



8-пазл — це головоломка, що складається з поля 3×3 з числами від 1 до 8 та однією порожньою клітинкою (0).  

Мета — отримати цільову конфігурацію шляхом пересування порожньої клітинки.



---



\## Використаний алгоритм



Алгоритм BFS (пошук в ширину:



\- Використовує структуру даних `queue` (чергу)

\- Переглядає стани рівень за рівнем

\- Гарантує знаходження найкоротшого шляху в незваженому графі

\- Використовує множину `visited` для уникнення повторів



---



\## Початковий та цільовий стани



\### Початковий стан:



(0, 7, 8)

(4, 5, 6)

(1, 2, 3)





\### Цільовий стан:



(1, 2, 3)

(4, 5, 6)

(7, 8, 0)
